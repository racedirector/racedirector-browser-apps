## About Displays the lateral and longitudinal acceleration on a scale of 3G on the drivers car. ## Setup 1. Go to [Settings](#!/settings) page 2. Copy generated **CLR Browser URL** 3. In OBS add new **CLR Browser** layer, and paste it in **URL** 4. Set **Dimensions** to 130x130px 5. Go to **CSS** tab and replace all with: ```css body { background: inherit !important; } ``` This will make background transparent
