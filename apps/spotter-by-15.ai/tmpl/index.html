## About Natural TTS spotter packs by [15.ai][1]. ## Install 1. Download zip file 2. Unzip file into `...Path_to_iRacing_install_folder/sound/spcc` It should look like this `E:\games\iRacing\sound\spcc\15.ai-sponge-bob\[bunch of files here]` 3. Load iRacing, go to Options &rarr; Sound &rarr; Voice Pack, choose Spotter. Set Chattiness to High. 4. restart iRacing ## THINGS TO REMEMBER 1. Remember to credit [15.ai][1] 2.
