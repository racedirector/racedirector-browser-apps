## About Helmet Overlay allows you to put an image of helmet while streaming in VR. [Video how it looks like](https://youtu.be/K5LfOIxL7BE). Idea, PSD template, and video by [Joachim Ljunggren](https://twitch.tv/TBR_Jocke). ## Install 1. Download and unzip [iRacingBrowserApps](/downloads/iRacingBrowserApps.zip) 2. Run `server.exe` 3. Install [OBS Studio](https://obsproject.com/download) (use **Full-Installer** with Browser plugin) ## Setup 1. Add **BrowserSource** layer in OBS Studio 2. Set **URL** to `https://ir-apps.kutu.ru/helmet-overlay/overlay.html` 3. Set **Width x Height** to your OBS Resolution (OBS Settings -> Video -> Output Resolution) 4. Set **FPS** to 20 5. You can change default helmet image to your own via CSS: ``` .overlay { background-image: url('http://url-to-image.jpg'); } ``` ## Customize 1. Download [Helmet Template](https://kutu182.narod.ru/ir-apps/helmet-overlay-template.psd) 2. Customize it in Photoshop 3. Upload PNG image to any image hosting service 4. Refer to "Setup" step 5 (make sure you use direct image url, not the url to a page with your image) ## For 2 PC streamers Use URL like this `http://ir-apps.kutu.ru/helmet-overlay/overlay.html?server=192.168.1.1:8182`
