## About OBS Stats - is a Browser friendly copy of **View &rarr; Stats** window in OBS Studio. ![Screenshot](img/screenshot.webp) ## Install 1. Install [OBS Studio](https://obsproject.com/download) 2. install [OBS-WebSocket](https://github.com/Palakis/obs-websocket/releases) ## Setup 1. Launch **OBS Studio** 2. Go to **Tools &rarr; Websockets Server Settings** 3. Input **Password** ## Usage `https://ir-apps.kutu.ru/obs-stats/obs-stats.html?address=localhost:4444&password=YOUR_PASSWORD_HERE`
