## About Twitch Chat - customizable twitch chat overlay for OBS. <p class="text-danger">**Warning**: You are not allowed to use this application, if you have any scrolling text in your overlay. #blamemancave</p> ## Install 1. Download and unzip [iRacingBrowserApps](/downloads/iRacingBrowserApps.zip) 2. Run `chat.exe` 3. Install [OBS Studio](https://obsproject.com/download) (use **Full-Installer** with Browser plugin) ## Setup 1. Go to [Settings](#!/settings) page 2. Copy generated **Browser URL** 3. Add **BrowserSource** layer in OBS Studio, and paste it in **URL** field 4. Set **Width x Height** to desired, don't scale layer itself in preview 5. Set **FPS** the same as your stream fps
